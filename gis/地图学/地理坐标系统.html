# 常用坐标系

# 北京54坐标系

BJ54（北京1954坐标系）是中国在1954年建立的大地坐标系统，它基于克拉索夫斯基椭球（Krasovsky ellipsoid）并广泛应用于中国的测绘和地理信息系统中，特别是在CGCS2000（中国大地坐标系2000）建立之前。

  1. 椭球参数

    • 长半轴（a）：6378245.0米
    • 扁率倒数（1/f）：298.3

  2. 投影方式

    • BJ54常使用高斯-克吕格（Gauss-Krüger）投影，将地理坐标转换为平面坐标。
    • 在高斯-克吕格投影中，中国通常使用6度分带。

  3. 参考大地原点

    • 大地原点在原苏联的普尔科沃

  4. 缺点

    • 长半轴大109m

    • 与我国大地水准面，自西向东有明显的系统倾斜，东部地区最大60m

# 西安80坐标系

西安80坐标系（Xian 1980 Coordinate System）是中国使用的一种坐标系统，主要用于中国大陆的地理和工程测量。以下是西安80坐标系的一些关键参数：

  1. 椭球参数

    • 椭球名称：西安80椭球（Xian 1980 Ellipsoid）

    • 长半轴（aaa）：6378140.0 米

    • 扁率（fff）：1/298.257

    • 第一偏心率的平方（e2e^2e2）：0.00669438002290

    • 第二偏心率的平方（e′2e'^2e′2）：0.00673949674228

  2. 投影方式

    • 投影方式：高斯-克吕格投影（Gauss-Krüger Projection）

    • 带宽：6度（通常用于中国大陆的国家大地坐标系统）

    • 中央子午线：每个投影带都有自己的中央子午线，带号通常用于定位。

  3. 参考大地原点

    • 陕西省泾阳县永乐镇，位于西安市西北方向约60公里。

  4. 差异

    • 长半轴比84坐标系大3m左右

# WGS84坐标系

WGS84（World Geodetic System 1984）是全球定位系统（GPS）所使用的地理坐标系统和地球参考框架。它是由美国国防部开发的，广泛应用于导航、制图和地理信息系统（GIS）中。

  1. 椭球参数
    • 长半轴（a）：6378137.0米
    • 扁率倒数（1/f）：298.257223563
    • 短半轴（b）：6356752.3142米
  2. 原点
    • 原点位于地球质心，（经纬度原点）在赤道和本初子午线的交点处。
  3. 坐标系统
    • 经度（Longitude）：从本初子午线向东或向西测量，范围为-180°到+180°。
    • 纬度（Latitude）：从赤道向北或向南测量，范围为-90°到+90°。
    • 高程（Elevation or Height）：相对于椭球面的高度。

# 国家2000坐标系

CGCS2000（China Geodetic Coordinate System 2000）是中国大地坐标系2000，是中国国家测绘地理信息局（NGCC）建立的地理坐标系，广泛应用于国家测绘、地理信息系统（GIS）和导航等领域。

  1. 参考椭球参数
    • 长半轴（a）：6378137.0米
    • 扁率倒数（1/f）：298.257222101
    • 地心引力常数　GM＝3.986004418×1014m3s-2
    • 自转角速度　ω＝7.292l15×10-5rad s-1
  2. 原点
    • 原点位于地球质心，（经纬度原点）参照国际大地参考框架（ITRF）。
  3. 坐标系统
    • 经度（Longitude）：从本初子午线向东或向西测量，范围为-180°到+180°。
    • 纬度（Latitude）：从赤道向北或向南测量，范围为-90°到+90°。
    • 高程（Elevation or Height）：相对于椭球面的高度。

# 常用高程系统

# 黄海高程基准

​ 黄海高程系是1956年确定的，它是根据青岛验潮站1950年到1956年的黄海验潮资料，求出该站验潮井里横按铜丝的高度为3.61 米，所以就确定这个铜丝以下3.61米处为黄海平均海水面。从这个平均海水面起，于1956年推算出青岛水准原点的高程为72.289米。

​ 黄海高程系位于山东青岛市大港1号码头西端青岛观象台的验潮站内，地理位置为东经120°18′40〃，北纬36°05′15〃。验潮站有一间特殊的房屋，内有一口直径1米、深10米的验潮井，它有3个直径60厘米的进水管与大海相通。

# 国家85高程基准

​ 中国测绘主管部门决定重新计算黄海平均海面，以青岛验潮站1952年～1979年的潮汐观测资料为计算依据，叫“1985国家高程基准”，并用精密水准测量位于青岛的中华人民共和国水准原点，得出1985年国家高程基准高程和1956年黄海高程的关系为：1985年国家高程基准高程=1956年黄海高程-0.029m。

​ 由于潮汐存在波长为19年的周期变化，所以高程基准应采用19年的观测数据进行计算。其实，1985国家高程基准就是这么计算来的。具体计算方法是：根据1952年～1979年的潮汐观测资料，计算时取19年的资料为一组, 滑动步长为1年，得到10组以19年为一个周期的平均海面, 取均值得到的结果作为黄海平均海水面，然后再推算出水准原点的高程。

# 各高程基准换算

  • 56黄海高程基准：+0.000

  • 85高程基准：56高程-0.029

  • 吴淞高程系统：56高程+1.688

  • 珠江高程基准：56高程-0.586

目前通用高程基准是：85高程基准

# Web开发坐标系

# GCJ02坐标系

GCJ-02（火星坐标系）是中国国家测绘局制定的一种地理坐标系统，用于在中国大陆范围内的地理数据，

  • 椭球参数与WGS84相同。

  • 坐标系基于WGS84坐标系进行了一定的加密和偏移处理，以保护国家地理信息安全。

  • 偏移算法是保密的，具体实现未公开。

# BD09坐标系

BD-09（百度坐标系）是百度公司推出的一种地理坐标系统。

  • 基于GCJ-02（火星坐标系）进行进一步的加密和偏移处理。
  • BD-09 的具体加密算法未公开，但有公开的第三方实现和逆向工程结果。

# Web墨卡托坐标系

Web墨卡托坐标系（Web Mercator Projection），也称为EPSG:3857，是一种广泛用于在线地图服务的投影坐标系。这种坐标系基于墨卡托投影，并经过了一些调整，以便适合互联网地图应用，如Google Maps、OpenStreetMap和Bing Maps。以下是Web墨卡托坐标系的关键特点和应用：

Web墨卡托坐标系的特点：

  1. 投影方式：
    • 投影类型：经纬度坐标的墨卡托投影（Mercator Projection），一种圆柱投影。
    • 投影参数：采用了伪圆柱投影，即将地球投影到一个圆柱上，然后展开这个圆柱。此投影方式会在极地区域造成变形，但对于中纬度区域表现较好。
  2. 坐标范围：
    • 经度范围：-180° 到 180°，由于投影的变形，实际上经度会被裁剪到大约-20037508.34 到 20037508.34米的范围。
    • 纬度范围：-85.0511° 到 85.0511°，这是因为墨卡托投影无法完整表示极地区域。
  3. 单位：
    • 单位：以米为单位，适合进行地图缩放和网格化处理。
  4. 标准化：
    • 参考椭球：Web墨卡托坐标系通常使用WGS84作为参考椭球，与GPS系统兼容。

# 各地图产品坐标系

为了保密，任何一个地图产品都不允许使用GPS坐标

天地图：CGCS2000

百度地图：BD-09坐标系，在GCJ-02的基础上进一步偏移得到

高德地图、腾讯地图、Google中国：GCJ-02坐标系，也称为火星坐标系

说明：

​ GCJ02相对于WGS84会向东北方向偏移，在滨海新区，其他地方不一定
