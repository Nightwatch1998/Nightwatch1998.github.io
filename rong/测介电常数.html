# 测介电常数

对机器的操作：将矢网打开，首先机器自己会打开一个默认的S参数软件，然后打开一个叫微波xx的软件，在里面修改频率（76.5GHz），修改倍频（改成6）；

校准方式选择哪一种：cal-smart cal-只勾选1-2端口——next——选择波导形式（WR12、W波段）——开始校准端口1和端口2，短路1端口，测试，短路2端口，测试，1、2端口直通，测试，1、2端口之间拉远1/4波长（空气波长1/4 -0.98mm，转动千分尺做调整），测试，完成校准；完成校准后，千分尺转回原来的1/4波长，矢网S21相位为0°左右，幅度为-30到-40dB，说明校准成功。

开始测试： 将待测外罩放在两端口之间，记录此时S21的相位w，通过这个相位就可以反推出该材料的介质参数。

如果测试的时候想要mark平均值，则使用avg bw——averging，调大这个数值，界面上显示的就是按照这个数值采数平均后的结果。

在读取相位的时候，保证S11的幅度不差（比较小）/S12的幅度最大 时，读取S12的相位。

计算公式：

校准后1-2端口之间相位为0度，待测外罩的厚度为h1，中心频点为f，空气波长为λ0=c/f，一个波长对应的2pi，λ0对应2pi（360°），于是h1对应的相位为ph1，λ0/360=h1/ph1，ph1=h1*360/λ0，这个ph1就是厚度为h1的空气经过的相位；

放入厚度为h1，介电常数为x的外罩后，1-2端口之间的相位从0变成了w，这个w就是把厚度h1的空气更换成厚度h1的外罩，厚度为h1的空气相位为ph1=h1360/λ0，厚度为h1的外罩相位为ph2=ph1+w=h1360/λ0+w，这个ph2对应穿过厚度h1介电常数为x的外罩的相位，即λ0/(根号下x)/360 = h1/ph2 = h1/(h1*360/λ0+w)，

根号下x = 360λ0(h1360/λ0+w)/h1 =1+wλ0/(h1360) ，w是矢网上的S21相位读数（角度制），λ0是空气波长（单位mm），h1是外罩厚度（单位mm）。 x = [1+wλ0/(h1*360)]²
