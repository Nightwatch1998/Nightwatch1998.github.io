# 开源项目学习

# Jeecg-Boot

Jeecg-Boot是一款基于BPM流程和代码生成的AI低代码平台，采用前后端分离架构，，强大代码生成器实现前后端一键生成，无需手写代码。

功能涵盖在线表单、表单设计、流程设计、门户设计、报表与大屏设计、OA办公、AI应用、AI知识库、大模型管理、AI流程编排、AI聊天，支持ChatGPT、DeepSeek、Ollama等多种AI大模型。

eecgBoot低代码平台兼容所有J2EE项目开发，支持信创国产化，特别适用于SAAS、企业信息管理系统（MIS）、内部办公系统（OA）、企业资源计划系统（ERP）、客户关系管理系统（CRM）及AI知识库等场景。

Gitee仓库地址 (opens new window)

文档地址 (opens new window)

当前最新版本：v3.8.3 2025年10月24日

使用Jeecg-Boot能够简化的工作内容：

  1. 网页创建数据库表结构，并且自动同步到数据库
  2. 代码生成：可根据表结构生成后端、前端、小程序代码
  3. 菜单、用户、角色、权限、部门管理
  4. 网页控制定时任务的启动、暂停
  5. 日志管理、日志直接存入数据库，网页查看
  6. 性能监控：服务器信息、JVM信息、Tomcat信息、磁盘监控、内存信息，很全面
  7. 设计报表、大屏、仪表盘
  8. 接入deepseekAI模型

用这个项目，想做一个简单的数据可视化系统非常快，开发可能只需要一周。

# 安装部署

1、从gitee上下载仓库

2、后端项目中，更改application-dev.yml中数据库和redis的连接方式

3、启动后端项目

4、前端项目执行pnpm install，
